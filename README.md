# 🌦️ Weather App

A simple weather application built using **HTML, CSS, and JavaScript** that fetches real-time weather information using the OpenWeatherMap API.

> 📌 This was one of my very first web development projects and marks the beginning of my journey into frontend development and working with APIs.

---

## ✨ Features

- 🌍 Search weather by city name
- 🌡️ Display current temperature
- ☁️ Weather description
- 💨 Wind speed information
- ⚡ Real-time weather data using OpenWeatherMap API

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla)
- OpenWeatherMap API

---

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/yourusername/weatherApp.git
```

2. Open `index.html` in your browser.

> **Note:** You'll need a valid OpenWeatherMap API key if you want to replace or update the existing one.
